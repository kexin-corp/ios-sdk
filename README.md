#  Kexin iOS SDK

## 简介
**为简化开发者的APP开发，可信提供了APP开发套件和相关的支持服务：**
- 提供APP的SDK和开发文档，兼容iOS、Android平以及HTML5嵌入式界
- 提供iOS端通用业务功能块的实例源代码，极大简化开发人员开发工作
- 为企业用户提供专业的技术支持服务

**SDK包含以下主要功能：**
- 自动扫描、发现可用设备锁
- 提供数据接收与发送的接口
- 提供底层网络和设备状态回调接口
- 使开发者简化APP开发流程，快速完成APP开发，开发者仅需关注APP业务功能即可，而相对复杂的协议与错误处理等事项可忽略。

## 版本要求
iOS 8 及以上版本
